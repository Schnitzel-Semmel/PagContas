# PagContas

PagContas e um sistema web para registrar gastos, organizar categorias e acompanhar contas a pagar de forma simples.

## Objetivo

O projeto foi pensado para facilitar o controle financeiro de usuarios que precisam de uma interface clara, direta e acessivel.

## Funcionalidades previstas

- Uso facilitado a partir do WhatsApp
- cadastro e login de usuarios
- registro feito por comando de voz ou audio
- categorias personalizadas
- contas com vencimento
- marcacao de gastos pagos e pendentes
- painel com resumo financeiro
- configuracoes de acessibilidade
- envio de relatorios em intervalos definidos pelo usuario

## Tecnologias

- HTML
- CSS
- JavaScript
- PHP
- MySQL
